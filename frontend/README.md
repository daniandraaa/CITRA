# CITRA Frontend

Frontend untuk aplikasi CITRA (CT Image Transform & Region Analyzer) yang dibangun menggunakan React, TypeScript, dan Vite.

## Fitur

- Antarmuka modern untuk unggah citra CT.
- Visualisasi hasil pemrosesan citra secara real-time.
- Kontrol parameter untuk windowing dan segmentasi.

## Cara Menjalankan

Pastikan Anda memiliki Node.js terinstal.

1. Install dependensi:
   ```powershell
   npm install
   ```

2. Jalankan aplikasi mode development:
   ```powershell
   npm run dev
   ```

Aplikasi akan tersedia di `http://localhost:5173`.

## Teknologi

- **React**: Library UI utama.
- **TypeScript**: Untuk keamanan tipe data.
- **Vite**: Build tool yang sangat cepat.
- **Tailwind CSS**: Untuk styling (jika digunakan).
